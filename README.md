# Ninja Creami Recipe Book

High-protein, low-calorie ice cream recipes for the Ninja Creami, designed for sustainable fat loss.

## Features

- **50+ recipes** with full nutritional information
- **JSON-LD schema markup** for Weight Watchers app import
- **Australian measurements** (metric, Celsius)
- **Adelaide shopping guide** included in recipes

## Recipe Categories

- **Vanilla Base** - Simple to restaurant-quality vanilla
- **Chocolate** - From basic to death by chocolate
- **Specialty** - Mint chip, cookie dough, cherry garcia, etc.
- **Cheesecake** - 6 tangy, creamy varieties
- **Sorbet** - ~100 calorie fruit sorbets
